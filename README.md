**P.League-Stats-Crawler**是一個工具可以得到 P.League 各類型統計資料。資料內容與 [P.League](https://pleagueofficial.com/) 網站提供的內容相同，但網站不提供下載的選項。因此P.League-Stats-Crawler透過爬蟲的技術，得到網站上的統計資料，提供給想要做資料分析的人可以有更快速的資料取得方式。
