## 前置作業
1. 下載到本機

		git clone https://github.com/HarryChenTw/PLeague-Stats-Downloader.git
2. 確認python版本

		python --version
   PLeague-Stats-Downloader是用3.8.8
3. 安裝相關套件

		pip install requirements.txt
